# i18n & RTL

## Translating the UI

Every user-visible string (ARIA labels, tooltips, the error message, the "Link copied" toast) is overridable via `labels`. Provide only the keys you want to change.

```ts
new Lightbox({
  items,
  labels: {
    close: 'Fermer',
    previous: 'Précédent',
    next: 'Suivant',
    zoomIn: 'Agrandir',
    zoomOut: 'Réduire',
    fullscreen: 'Plein écran',
    slideshowStart: 'Diaporama',
    slideshowPause: 'Pause',
    download: 'Télécharger',
    share: 'Partager',
    error: 'Impossible de charger ce média',
    linkCopied: 'Lien copié',
  },
})
```

Full key list:

```ts
interface LightboxLabels {
  dialog; close; previous; next
  zoomIn; zoomOut; fullscreen
  slideshowStart; slideshowPause
  download; share
  rotateLeft; rotateRight; flipHorizontal; flipVertical
  thumbnails; slide; error; linkCopied
}
```

The English defaults are exported if you want to extend them:

```ts
import { DEFAULT_LABELS } from '@anil-labs/lightbox-gallery-core'
```

## Right-to-left

RTL is auto-detected from `document.documentElement.dir`, or set it explicitly:

```ts
new Lightbox({ items, rtl: true })
```

In RTL mode the previous/next buttons swap sides and the arrow keys reverse, so navigation matches reading direction.
