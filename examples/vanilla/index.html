Every feature, one page. Each card is a self-contained example — find the one that matches your use case.

Basic Mixed media Slideshow Toolbar Themes Inline Deep-linking API & events

01 Basic gallery

bindGallery() turns a set of links into a gallery — click a thumbnail and it opens with a FLIP zoom from that image. Wheel/pinch zoom, swipe, arrows, f, Esc all work out of the box.

Use it for portfolios, product shots, any grid of photos.

02 Mixed media

One gallery, many types — images, an MP4 video, a YouTube embed and a raw HTML slide. Types are auto-detected from the URL, or set explicitly with type.

Use it for mixed image + video galleries, case studies, press kits.

03 Slideshow

Autoplay with a progress bar (slideshow, slideshowDelay, slideshowProgress). Hovering the stage pauses it.

Use it for kiosks, hero showcases, hands-off presentations.

04 Toolbar & actions

Opt into fullscreen, download, share and rotate buttons — plus your own via toolbarButtons.

Use it for asset libraries, downloadable galleries, admin tools.

05 Themes

theme: 'dark' | 'light' | 'auto' plus the lbg-theme-glass and lbg-theme-minimal presets (via className). Everything is CSS custom properties, so you can brand it further.

Use it for matching your site's look, light pages, OS-aware UIs.

06 Inline carousel

inline: true with a container renders the viewer right on the page instead of as a fullscreen overlay — a zoomable, swipeable carousel.

Use it for product pages, embedded galleries, dashboards.

07 Deep-linking (URL hash)

hash: true writes the open slide to the URL (#gallery=3), so links are shareable and the browser back button closes the viewer. Open one, then check the address bar.

Use it for shareable photos, SEO-friendly galleries, back-button UX.

08 Programmatic API & live events

Drive the instance from code — open, next, prev, toggleSlideshow, toggleFullscreen — and watch every event fire.

Use it for custom controls, analytics, syncing state elsewhere.
